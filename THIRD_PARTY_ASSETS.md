# Ressources tierces

## Tuffy

`Data/Fonts/tuffy.ttf` est la police Tuffy 1.1 de Thatcher Ulrich, distribuée sous CC0 dans les exemples SFML 2.6.2. Elle remplace la police implicite fournie par SFML 1.x, qui n'existe plus dans SFML 2.

- Source : <https://github.com/SFML/SFML/tree/2.6.x/examples/joystick/resources>
- Informations de licence SFML : <https://github.com/SFML/SFML/blob/2.6.x/examples/asset_licenses.md>
